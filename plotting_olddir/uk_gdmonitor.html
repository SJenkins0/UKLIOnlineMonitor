* This page contains monitoring plots made using data from the UK light injection system using LEDs of wavelength 435nm.

* B4 injector position is at z= -676.65 m / B5 injector position is at z = -1312.95

* For any problems, contact lanthony@ic.ac.uk and jmmcelwee1@sheffield.ac.uk

3. Figures

* Current plots shown are collimator and diffuser plots over 6h and 24h.

* Plots currently include those for total charge in SK for all collimators and diffuser. Plots for prompt charge inside the beam spot for both collimator and diffuser sources are not yet uploaded.

* Currently no uncertainty on data points. This is coming soon.

* For 24h plots, there are 30 subruns for each data point. For 6h plots, there are 6 subruns per data point.
